Este repositorio contiene la solución de dos ejercicios de HackerRank desarrollados para el examen práctico.

Los ejercicios seleccionados pertenecen a las categorías String y Search.

1: Strong Password
El problema consiste en determinar la cantidad mínima de caracteres que deben agregarse a una contraseña para que cumpla con los requisitos de seguridad establecidos:

- Al menos un número.
- Al menos una letra minúscula.
- Al menos una letra mayúscula.
- Al menos un carácter especial.
- Longitud mínima de 6 caracteres.
  
Complejidad temporal: O(n)
Complejidad espacial: O(1)

2: Intro to Tutorial Challenges

El problema consiste en encontrar la posición de un valor dentro de un arreglo.

La solución realiza una búsqueda secuencial recorriendo el arreglo hasta encontrar el valor.

Complejidad temporal: O(n)
Complejidad espacial: O(1)


Compilación

g++ strong_password.cpp -o strong_password

g++ intro_tutorial.cpp -o intro_tutorial

Ejecución

./strong_password

./intro_tutorial
